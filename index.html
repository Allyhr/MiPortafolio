Sobre mí

¡Hola! Soy Allison Lia Huerta Ramírez, Ingeniera en Sistemas Computacionales.

Mi formación comenzó desde el nivel medio superior, donde cursé una carrera técnica en informática, lo que me permitió adquirir una base sólida en tecnologías de la información desde temprana etapa.

Algo que me ha enriquecido mucho es haber colaborado en todas las fases del ciclo de desarrollo: documentación, diseño de interfaces, modelado de bases de datos, programación, pruebas (testing), control de versiones y liderazgo de proyectos.

También he tenido la oportunidad de adquirir conocimientos en administración y contabilidad, lo cual me ha permitido desarrollar una mejor comprensión del entorno empresarial y complementar mi perfil con una perspectiva organizacional.

En este portafolio comparto algunos de los proyectos que he realizado, y estoy abierta a nuevas oportunidades donde pueda aportar mis habilidades y seguir creciendo profesionalmente.

Mis Proyectos

Sistema de Gestión Veterinaria

Sistema desarrollado para la gestión de clínicas veterinarias. Cuenta con dos tipos de usuario: veterinario y auxiliar, cada uno con permisos y vistas específicas. Permite registrar, modificar y eliminar información tanto de personal como de pacientes (mascotas), así como llevar un historial detallado de consultas.

  • Gestión de usuarios: veterinarios y auxiliares
  • Registro, edición y eliminación de mascotas
  • Historial médico de cada mascota con observaciones, medicamentos, exámenes y padecimientos
  • Sección de estadísticas gráficas sobre padecimientos más comunes
  • Permisos diferenciados por tipo de usuario
Java MySQL NetBeans / IntelliJ IDEA XAMPP

Sistema de Monitoreo y Automatización para Invernadero

Aplicación web integrada con hardware mediante Arduino. Permite monitorear en tiempo real las condiciones del invernadero (temperatura y humedad), y automatiza el riego según los niveles detectados.

  • Actualización automática de datos en una tabla cada 2 segundos
  • Dos perfiles de usuario: administrador (gestión de usuarios y visualización de registros) y trabajador (solo lectura)
  • Integración con sensores de temperatura y humedad, bomba de agua, display LCD, botón y potenciómetro
  • Activación automática de la bomba según temperatura detectada, o manual mediante botón
  • Visualización de condiciones en pantalla LCD y almacenamiento en MongoDB
HTML Python JavaScript CSS MongoDB Arduino

Sistema Web de Historial Académico

Sistema web que permite la gestión y consulta del historial académico de los estudiantes. Cuenta con tres perfiles de usuario: administrador, personal de control escolar y alumno, cada uno con permisos y vistas específicas.

  • Administrador: gestiona usuarios, perfiles y módulos del sistema (activar/desactivar, modificar, eliminar), y visualiza la actividad de otros usuarios
  • Personal de control escolar: administra carreras, especialidades, materias, retículas, estudiantes y calificaciones
  • Alumno: puede consultar, imprimir o descargar su historial académico sin necesidad de acudir a control escolar
  • Control de accesos por rol y estatus de vista (activo/inactivo)
HTML PHP JavaScript MySQL Visual Studio Code XAMPP

Sistema de Gestión para el Área de Titulación

Aplicación de escritorio diseñado para el área de titulación de una universidad. Permite automatizar la generación de certificados de estudios totales a partir de archivos Excel con calificaciones.

  • Gestión de carreras, especialidades, materias y plantillas de certificados
  • Funcionalidad para subir archivos Excel con calificaciones
  • Validación automática del formato del archivo Excel
  • Identificación de alumno, carrera y plantilla correspondiente
  • Mapeo automático de las materias con sus calificaciones hacia la plantilla de certificado
  • Generación masiva de certificados de estudios totales
JavaFX MySQL Apache POI (para manejo de Excel) IntelliJ IDEA XAMPP

Sistema de Gestión para Cría de Tenebrio Molitor

Sistema de escritorio que permite gestionar unidades de cría de Tenebrio molitor, utilizado en procesos de entomocultura industrial. Automatiza el seguimiento de condiciones clave como alimentación, limpieza y variables ambientales, mejorando la eficiencia y precisión del manejo.

  • Registro, edición y eliminación de unidades de cría
  • Captura de datos como dimensiones, biomasa, densidad, técnica utilizada, ubicación y condiciones ambientales (temperatura, humedad, oxigenación)
  • Cálculo automático de volumen y densidad
  • Notificaciones programadas para alimentación, limpieza y cambios de estado
  • Interfaz con barra de navegación: Inicio, Configurar, Registrar usuario, Cerrar sesión
  • Configuración personalizada del intervalo de notificaciones
Python MySQL Visual Studio Code XAMPP
